Подключение к базе данных:
	Перед тем, как запускать Picker, необходимо зашифровать пароль, он будет работать только на этом компьютере для любого пользователя.
	Чтобы зашифровать пароль, нужно запустить отдельное консольное приложение EncryptPass, далее выбрать вариант 2, после этого ввести пароль.
	Далее из папки копируем текстовый файл pass.txt и вставляем его в папку с приложением Picker.

	Строка подключения может редактироваться из appsettings.json, не пытайтесь обойти систему расшифровки пароля, вставляя его в строку подключения,
	в этом случае к базе данных вы не подключитесь.

База данных:
	В приложении пока что возможно использование только MySql и только версии 5.5.39

Использование приложения:
	Включая и отключая чекбоксы, Вы можете сохранять в базу данны различное сочетание полей:
		1)Вопрос
		2)Вопрос + 2 ответа
		3)Вопрос + 2 телефона
		4)Одно время (либо начало, либо конец опроса)
		5)Весь интервал (и начало и конец)
		6)Все поля
	Будет включаться проверка на корректность данных для каждого из полей.
	Если после нажатия на кнопку "сохранить" ничего не произошло - значит такое сочетание не реализовано.
	
	Когда внизу горит зеленая лампочка,это означает что есть подключение к базе данных.
	Если она горит красным - значит подключения нет.
	Эта проверка не происходит постоянно, но вы можете нажать на лампочку и она произойдет.

	Программа подразумевает сохранение одинаковых значений полей. Никакого редактирования созданных строчек в базе данных через эту программу не подразумевается.


Классы для фронта:
	ClockContext - Три поля ObservableCollection<ITimeItem> Houritems, Minitems, Secitems
	используются для заполнения сответствующих ComboBox.
	SaveDataContext - контекст прослойка, которая связывает данные из полей интерфейса с моделью Vote для сохранения в базу данных.
		Также происходит валидация данных. результат возвращается через строчку.
		сигнатуры методов: bool CheckConn() - проверка подключения к бд, true если есть подключение, false - если нет.
							string SaveData(string) - сохранение вопроса					
							string SaveData(string,string,string) - сохранение вопроса и двух ответов
							string SaveData(string,string,string,string,string) - сохранение вопроса, двух ответов и двух телефонов
							string SaveData(ITimeItem,bool) - сохранение одного времени, true для vote_start и false для vote_stop
							string SaveData(ITimeItem,ITimeItem,ITimeItem,ITimeItem,ITimeItem,ITimeItem) - сохранение интервала времени
							string SaveData(string,string,string,ITimeItem,ITimeItem,ITimeItem,ITimeItem,ITimeItem,ITimeItem,string,string) - сохранение всех полей
	Остальные классы из библиотеки Pickerlib не используются во фронте.

	ElementsAction - класс отвечающий за методы связанные с элементами интерфейса.
					сигнатуры методов: Bitmap SetSource(bool connection) - Настройка цвета кнопки подключения. В метод передается результат проверки подключения,
																и исходя из значения возвращается соответвенно урл зеленой или красной кнопки. кнопки находятся в папке Resources,
																их названия - button green.png, button red.png
										
										CheckedAction(bool param, object sender, Grid maingrid) - Активация/Дизактивация полей для ввода данных
																									первый аргумент - значение чекбокса, второй аргумент - сам чекбокс
																									третий - табоица компоновки всех элементов.
										
										string SetValue(bool? check, TextBox element) - установка значения для строчного поля
										
										ITimeItem SetValue(bool? check, ComboBox element) - установка значения для интервального поля
										
										bool MakeRegion(ComboBox hourbox1, ComboBox minbox1, ComboBox secbox1, ComboBox hourbox2, ComboBox minbox2, ComboBox secbox2) -
										метод проверяющий правильность заданного интервала, если начало опроса будет позже конца - возврат false;
